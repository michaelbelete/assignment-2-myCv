  • Does it matter to use relative size(%,em,rem) one styling the size of an element so that the element is responsive ? why ?
    Yes having relative size make the contents in the website scale up and down while there is change in resoultion. since the size of the elements is relative to the device size. like if we consider the rem and em
    • Em is always relative to root element(html).
    • Rem is always relative to the parent element.
    so, rem, % and em size is not fixed it changes while the width and height of the device changes
  • We said 16px is default browser width so what does it mean when we say 200px ? How does the browser interprets that ?
    yes 16px is default size but when we introduce a new size like 200px the browser override the default 16px and uses the new size which in this case is 200px
